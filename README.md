# 2026新年烟花祝福网页

一个精美的2026新年烟花祝福网页，包含动态烟花效果、祝福文字系统和音效功能。

## 🎆 功能特点

- **动态烟花效果**：多定时器持续发射烟花，点击屏幕可触发额外烟花
- **祝福文字系统**：支持自定义输入祝福文字，或点击快捷祝福按钮
- **漂浮文字动画**：祝福文字以动画效果漂浮显示并动态变色
- **音效系统**：包含发射和爆炸音效，可开关
- **响应式设计**：自适应不同屏幕尺寸
- **视觉特效**：渐变背景、发光效果、粒子动画等

## 🛠️ 技术栈

- **HTML5**：页面结构
- **CSS3**：样式和动画效果
- **JavaScript**：交互逻辑和动画
- **jQuery**：DOM操作和事件处理
- **Canvas API**：烟花粒子动画
- **Web Audio API**：音效系统

## 🎯 核心功能实现

### 1. 烟花动画系统

#### 火箭类 (Rocket)
- 从屏幕底部发射到随机高度
- 到达目标高度或速度为正时爆炸
- 爆炸时产生粒子效果

#### 粒子类 (Particle)
- 爆炸后向四周扩散
- 受到重力影响下落
- 透明度逐渐降低直至消失

#### 动画循环
- 使用 `requestAnimationFrame` 实现流畅动画
- 半透明覆盖产生拖尾效果
- 持续清理过期粒子，优化性能

### 2. 音效系统

- **发射音效**：上升的正弦波音效，频率从200Hz上升到800Hz
- **爆炸音效**：结合噪音和低频振荡器，模拟真实爆炸声音
- **音频上下文管理**：自动处理浏览器音频策略，点击页面初始化音频
- **音效开关**：右上角按钮控制音效开启/关闭

### 3. 祝福文字系统

- **自定义输入**：底部输入框支持自定义祝福文字
- **快捷祝福**：预设多种常用新年祝福按钮
- **漂浮动画**：文字以随机位置、大小、角度和颜色漂浮显示
- **动态变色**：文字颜色随时间动态变化

## 🚀 使用方法

1. **打开网页**：在浏览器中打开 `index.html` 文件
2. **查看烟花**：页面自动播放烟花效果，可点击屏幕触发更多烟花
3. **发送祝福**：
   - 在底部输入框输入祝福文字，点击"🎆 发射祝福"按钮
   - 或点击上方快捷祝福按钮
4. **控制音效**：点击右上角"🔊 音效"按钮开关音效
